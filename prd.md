# 国家队持仓追踪看板 — PRD

## 1. 产品目标

构建一个 Streamlit Web 看板，展示国家队（社保基金、中央汇金、国新投资等）的 A 股持仓变化，帮助投资者快速发现加仓/减仓信号。

## 2. 技术方案

| 组件 | 选型 | 说明 |
|------|------|------|
| 前端框架 | Streamlit | Python 全栈，快速开发 |
| 托管 | Streamlit Community Cloud | 免费，连接 GitHub 仓库自动部署 |
| 定时爬取 | GitHub Actions (cron) | 每天凌晨跑爬虫，结果 commit 回仓库 |
| 数据存储 | CSV in repo | 轻量，Streamlit Cloud 直接读取 |

### 为什么不用 GitHub Pages

GitHub Pages 只支持静态文件。Streamlit 需要 Python 运行时，Streamlit Community Cloud 免费提供这个能力，且支持从 GitHub 仓库自动部署和更新。

## 3. 页面设计

### 3.1 首页 — 持仓变化总览

- 顶部：季度选择器（下拉框，默认最新两个完整季度对比）
- KPI 卡片行：新进股票数 / 加仓股票数 / 减仓股票数 / 退出股票数 / 净加仓金额
- 加仓 TOP 表格：股票代码、股票简称、机构数变化、流通市值变化(亿)、当前持有机构数、持有机构列表
  - 支持排序（按市值变化/机构数变化）
  - 支持搜索（按股票代码或简称筛选）
- 减仓 TOP 表格：同上结构
- 新进股票表格 / 退出股票表格（折叠展示）

### 3.2 个股详情页

- 顶部：股票搜索框（代码或简称，支持模糊搜索）
- 持仓时间线：折线图，X 轴为季度，Y 轴为国家队持仓总流通市值
- 持有机构变化：堆叠柱状图，每个季度各机构的持仓金额
- 持仓明细表：每个季度的持有机构、数量、流通市值

### 3.3 国家队一览页

- 左侧：国家队机构列表（按当前持仓总市值排序）
- 右侧：选中机构后展示
  - 当前持仓股票列表（代码、简称、数量、流通市值、报告期）
  - 持仓变化：本季 vs 上季的增减
  - 持仓集中度：行业分布饼图（如有行业数据）

### 3.4 季度全景页

- 季度选择器
- 热力图/气泡图：X 轴为行业，Y 轴为机构数变化，气泡大小为市值变化
- 行业加仓排行：按行业汇总的净加仓金额柱状图
- 数据表：该季度所有持仓股票的完整列表，支持导出 CSV

## 4. 数据管道

### 4.1 GitHub Actions 定时任务

```
每天 UTC 16:00（北京时间 00:00）触发
  → 启动 Chrome headless 爬虫
  → 增量爬取 selenium_country_team_stock.csv
  → 生成透视表和分析 CSV
  → git commit + push 回仓库
  → Streamlit Cloud 自动检测到更新并重新加载
```

### 4.2 数据文件

| 文件 | 用途 | 更新频率 |
|------|------|----------|
| data/raw/selenium_country_team_stock.csv | 原始持仓数据 | 每日增量 |
| data/processed/quarterly_change_analysis.csv | 季度变化分析 | 每日重新生成 |
| data/processed/quarterly_stock_statistics.csv | 季度持仓透视表 | 每日重新生成 |
| data/config/stock_holder.csv | 股东实体配置 | 手动维护 |

## 5. 验收标准

验收方式：本地启动 Streamlit 服务，用 Puppeteer 自动化访问页面并截图验证。

### AC-1: 首页加载与 KPI 展示
- 访问首页，页面在 10 秒内加载完成
- 可见 KPI 卡片（新进/加仓/减仓/退出数量）
- 加仓 TOP 表格至少显示 10 行数据
- 截图验证布局正确

### AC-2: 季度切换
- 切换季度选择器到不同季度
- KPI 数字和表格数据随之更新
- 表格内容与该季度的实际数据一致

### AC-3: 个股搜索与详情
- 在个股详情页搜索 "紫金矿业" 或 "601899"
- 显示持仓时间线折线图
- 显示持有机构变化柱状图
- 持仓明细表包含正确的季度数据

### AC-4: 国家队一览
- 机构列表按持仓总市值排序展示
- 点击 "中央汇金资产管理有限责任公司"
- 右侧展示其当前持仓股票列表
- 持仓变化（本季 vs 上季）正确显示

### AC-5: 季度全景
- 选择一个季度后，行业加仓排行柱状图正确渲染
- 数据表包含该季度所有持仓股票
- 导出 CSV 功能可用

### AC-6: GitHub Actions 爬虫流水线
- .github/workflows/crawl.yml 配置正确
- 手动触发 workflow 能成功运行爬虫
- 爬取结果自动 commit 回仓库

### AC-7: Puppeteer 自动化验收脚本
- 运行 `node tests/e2e/verify.js`
- 脚本自动启动 Streamlit → 打开浏览器 → 逐页截图
- 截图保存到 tests/e2e/screenshots/
- 脚本退出码 0 表示所有页面加载成功
